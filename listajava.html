O que é JavaScript?

O JavaScript foi criado em 1995 por Brendan Eich para adicionar interatividade às páginas web. Ele surgiu com o objetivo de tornar os sites mais dinâmicos e responsivos, diferente do HTML e CSS, que são estáticos.

A principal diferença de execução está no ambiente: JavaScript no navegador é executado no lado do cliente (frontend), enquanto no servidor é possível executar JavaScript usando plataformas como o Node.js (backend).

No navegador, o JavaScript funciona dentro de um ambiente seguro chamado sandbox, que impede o acesso direto ao sistema do usuário, garantindo proteção contra códigos maliciosos.

Entre os objetos globais do JavaScript, destacam-se:
document – permite manipular a estrutura HTML da página.
console – exibe mensagens de depuração no console do navegador.
alert – mostra caixas de diálogo com mensagens para o usuário.
